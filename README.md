# LifeMate - Family Daily Planner

A real PWA (Progressive Web App) with Firebase backend.

## Features
- ✅ Full persistence — settings saved forever
- ✅ Household sync — family shares shopping list & todos in real time
- ✅ Password protected
- ✅ Works as a real app on iPhone (Add to Home Screen)
- ✅ Medicine reminders, sleep reminders, streaks

## Tech Stack
- React
- Firebase Firestore (database)
- Vercel (hosting)

## Setup
Already configured and ready to deploy to Vercel.
